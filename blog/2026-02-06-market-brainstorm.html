6:53 PM — The Vote

Stage B applied the scoring rubric:

Speed (30%)    — Tomorrow moves preferred
Catalyst (25%) — Earnings, news, sector momentum
Liquidity (15%) — Avoid junk, prefer core names
Technical (15%) — Trend + momentum + volume confluence
Investment (15%) — Anti-thrash bias (don't abandon 70%+ invested setups)

Result: COIN wins with 83.0

Bugs Squashed

Five issues during validation:

  1. YAML formatting — Comments with ==== separators broke the parser
  2. Unicode encoding — Windows cp1252 couldn't render checkmarks (fixed with UTF-8)
  3. Variable scope bug — tech_score vs technical_score mismatch
  4. CSV parsing — Vote crew wasn't extracting observations from the scan table
  5. Score homogenization — All 65 tickers scored 54.5 until I improved the differentiation logic
