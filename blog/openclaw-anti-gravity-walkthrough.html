1. Prerequisites (Do These First)

  • Node.js 22+
  • npm (comes with Node)
  • Google account (free)
  • Mac / Windows / Linux

Install Anti-Gravity

Download and install Anti-Gravity for your OS from their official site. This is Google's agentic IDE used as your execution layer.

2. Install OpenClaw Globally

Open a terminal / command prompt:

npm install -g openclaw

This installs OpenClaw system-wide.

3. Read the Security Warning (Important)

OpenClaw has full system access capabilities.

Best practice (recommended): Install inside a VM or sandbox, restrict filesystem permissions, docs before production use.

The video author did not sandbox (and explicitly says not to copy that mistake).

4. Enable Anti-Gravity OAuth in OpenClaw

This is the key step that unlocks free powerful models.

Step 4.1

Enable the Plugin

From the OpenClaw docs (linked in the video description), copy the command to enable the Anti-Gravity plugin and Google OAuth support. Paste it into your terminal and run it.

5. Log In With Your Google Account

Run the login command provided in the docs:

openclaw login

Browser opens. Log into your Google account. OAuth credentials are stored locally.

At this point, OpenClaw can use Google-backed models for free. This method is officially supported ‚Äî Google has publicly acknowledged it.

6. Run OpenClaw Onboarding

Now initialize everything else:

openclaw onboard

During onboarding:

Step 6.1

Acknowledge Security Warning

Read it. Confirm you understand the risk.

Step 6.2

Choose Setup Type

Select Quick Start

Step 6.3

OAuth Provider

Keep existing Anti-Gravity OAuth (don't overwrite)

Step 6.4

Model Provider

Select Google Anti-Gravity

Step 6.5

Model Selection

Pick one:

  • Opus 4.5 (thinking)
  • Gemini variants
  • Any available free model
Step 6.6

Messaging Gateway (Optional)

Choose if you want OpenClaw accessible from:

  • WhatsApp
  • Telegram
  • Discord
  • Slack

You can skip this and add later.

Step 6.7

Skills & Hooks

Skip for now (you'll enable manually later)

7. Open the OpenClaw Dashboard

After onboarding, the dashboard auto-launches. If it doesn't:

openclaw dashboard

This opens at: http://localhost:

8. Enable Core Skills

Inside the Dashboard, go to Skills tab. Recommended to enable:

  • Coding agent (OpenClaw Codex)
  • Planning / orchestration tools
  • Execution routing

This lets OpenClaw decide which agent does what.

9. Install the Anti-Gravity Agent Kit

This adds specialist agents:

  • UI / UX Designer
  • Builder
  • Reviewer
  • Debugger
npm install -g anti-gravity-kit

Now OpenClaw can call these agents automatically.

10. Install Community Skills via Claw Hub

Claw Hub = open-source skill marketplace. Example installs use npx:

npx clawhub install <skill-name>

Useful skills:

  • Anti-Gravity quota checker (see remaining free usage)
  • Image generation agents
  • Workflow automation helpers

11. Verify the Active Model

Ask OpenClaw:

"What model are you using?"

You should see something like:

Opus 4.5 via Anti-Gravity OAuth
Gemini variants

This confirms everything is wired correctly.

12. Using OpenClaw (3 Ways)

Option 1: Dashboard (Visual Control)

  • Run agents
  • Trigger skills
  • Monitor sessions
  • Schedule cron jobs

Option 2: Terminal

openclaw
# or

Option 3: Messaging Apps (if enabled)

  • WhatsApp
  • Telegram
  • Discord

You can literally deploy apps from your phone.

13. Real Workflow Example

  1. Prompt OpenClaw:
    "Build me a CRM dashboard"
  2. OpenClaw: Plans task, routes execution to Anti-Gravity agents
  3. Anti-Gravity:
    • UI agent designs interface
    • Builder writes code
    • Reviewer refines output
  4. Result:
    • Fully generated frontend
    • Iterated & refined
    • Deployed locally

Second pass ‚Üí cleaner UI, fewer errors.
